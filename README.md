# GrootanClient

A web tool to import csv files.

## Dependencies

`GrootanClient` is a basic html web application.

## Installation

1. `git clone git@github.com:Gopikapayoor/GrootanServer.git`
1. `cd GrootanServer`
1. `npm install`
1. `npm start`
1. Open web browser at url `http://localhost:1234`
